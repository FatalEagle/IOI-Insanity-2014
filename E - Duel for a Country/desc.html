Maniacal Midsummer Marathon 2014 by TL, AL, JJ

Problem E: Duel for a Country

Shiro is enthralled by the latest FPS (first person shooter) game in the Eastern Federation. A lot is riding on this game, so she is planning to snipe the other player, Izuna, for a decisive victory. Shiro plans to pull this off from a kilometre away, and Izuna is in a shelter, so this requires very advanced mathematical calculations. Shiro needs a special sequence of numbers, the K-numbers to accomplish her goal.

The K-numbers numbers have an interesting property: if you write them in base 10, and you find that the representation is dp × 10 p + dp - 1 × 10 p - 1 + … + d0 × 10 0 for decimal digits di (0 ≤ i ≤ p) and some nonnegative integer p, then there will be a subsequence of {dp, dp - 1, …, d0} that has a length of at least 6 and is in nondecreasing order. More formally, there exists a set {c0, c1, …, ck} where k ≥ 5 and 0 ≤ ci ≤ p for every i such that 0 ≤ i ≤ k such that cx > cx + 1 and dcx ≤ dcx + 1 for every x such that 0 ≤ x ≤ k - 1.

Shiro needs all the K-numbers between A and B (1 ≤ A ≤ B < 1018) to complete her calculations. In the time you need to compute all those numbers, she is sure that she can calculate the exact numbers in her head. Therefore, Shiro only needs you to determine the number of K-numbers that exist between A and B (to verify her calculations).

Input Format

The first line will have A. The second line will have B. You can assume that 30% of the time, Shiro will only give you A and B such that 1 ≤ A ≤ B < 107.

Output Format

A single integer, the number of K-numbers between A and B (inclusive).

Sample Input 1

111111
111131

Sample Output 1

17

Explanation for Sample 1

The valid K-numbers are 111111, 111112, 111113, 111114, 111115, 111116, 111117, 111118, 111119, 111122, 111123, 111124, 111125, 111126, 111127, 111128, 111129.

Sample Input 2

2454449
2454455

Sample Output 2

2

Explanation for Sample 2

The valid K-numbers are 2454449 and 2454455.

Sample Input 3

1111111
1111123

Sample Output 3

12

Explanation for Sample 3

Note that 1111120 is not a K-number, but 1111121 is.
